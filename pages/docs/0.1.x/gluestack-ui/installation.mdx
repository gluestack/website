# Installation

`gluestack/ui` is in the early preview.

### gluestack/ui is the default UI library in gluestack projects

`gluestack/ui` comes pre-packed with the **Web** & **Mobile** plugins of gluestack, which are Next.js and Expo projects respectively.

### Installing in a Next.js project

We don’t officially support gluestack/ui in any Next.js project.

But if you still want to, you can install Expo and then

```bash
npm install @gluestack/ui --save
```

### Installing in a React Native / Expo project

We don’t officially support gluestack/ui in any React Native / Expo project.

But if you still want to, you can install Expo and then

```bash
npm install @gluestack/ui --save
```
