# How to install

##

<br/>

* First go to your local terminal and type this command in order to install this plugin
  <br/>
  
    ```bash
    $ node glue add storage storage
    ```
     The **Storage plugin instance** name should only be **'storage'**.

  <br/>

* Now, after installing the dependency it will ask you for selecting the storage instance and
  graphql instance. Select the instance option with which you want to connect your storage
  plugin.

  Option will be something like this:
