# How to Install

##

<br/>

* To add Storybook into your gluestack project, you can run the following command in project's root directory:

```sh
$ node glue add storybook <instance-name>
```

* For instance, let's install an instance named **storybook** in our gluestack project:
```sh
$ node glue add storybook storybook
```

<br/>


* If the plugin is installed successfully, you can see a new folder named **storybook** has been created.
