


# Goals

##

<br/>

* Include **Authentication** mechanism into your application.

<br/>

* User can **login/signup manually**.

<br/>

* User can login/signup via social platforms like **Google,**
  **Microsoft, and Github**.

<br/>

* If the user logins successfully it should generate **jwt token**
  and send it as a response to the **Client**, using which they can
  **Authenticate** users.
