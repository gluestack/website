# How to install

##

<br/>

* To add PG-Admin into your gluestack project, you can run the following command in project's root directory:

```sh
$ node glue add pg-admin <instance-name>
```

* For instance, let's install an instance named **pg-admin** in our gluestack project:
```sh
$ node glue add pg-admin pg-admin
```

* After that, you will be prompted with following question -
