# Getting Started

##

<br />

The **nodemailer** plugin adds a **nodemailer** micro-service as an instance to send-out emails from your gluestack project.

With this plugin, you can easily integrate email sending functionality into gluestack your project. The plugin provides a simple and efficient way to configure your email settings and start sending emails right away.
