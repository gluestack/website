# Getting Started

##

Glue Plugin Engine is the process manager for the gluestack project.

This plugin helps you in managing all the installed **plugin instances** which you
have created in your **Gluestack App**, and in order to do that it comes with various commands
detailed in the **CLI Reference** section.
